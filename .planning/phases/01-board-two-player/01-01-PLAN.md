<read_first>

<action>
Edit `app/main.py` to add static file serving. Add these imports at the top alongside
existing imports:

```python
from fastapi.staticfiles import StaticFiles
from fastapi.responses import FileResponse
import os
```

After the `app = FastAPI(...)` block (but BEFORE any route definitions), add:

```python
# Serve static assets (CSS, JS, images) from the static/ directory.
# The "static" path prefix means files are available at /static/filename.
_static_dir = os.path.join(os.path.dirname(__file__), "..", "static")
app.mount("/static", StaticFiles(directory=_static_dir), name="static")
```

Then add a root route that serves index.html. Place it AFTER the mount but BEFORE the
existing `/health` route:

```python
@app.get("/", include_in_schema=False)
def serve_frontend():
    """Serve the game UI at the root URL."""
    return FileResponse(os.path.join(_static_dir, "index.html"))
```

Do NOT add any CORS middleware — per INT-03, it's not needed (same origin).

Verify existing tests still pass — the new route and mount should not affect existing
API endpoint behavior.
</action>
